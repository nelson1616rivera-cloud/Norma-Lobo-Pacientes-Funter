# Attendance Registration System

## Project Description
This is a simple attendance registration system that allows users to register their attendance using a web interface built with HTML, CSS, JavaScript, and a backend powered by Node.js, Express, and SQLite.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/nelson1616rivera-cloud/Norma-Lobo-Pacientes-Funter.git
   cd Norma-Lobo-Pacientes-Funter
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Set up the SQLite database (refer to the database setup instructions in the documentation).
4. Run the application:
   ```bash
   npm start
   ```

## Usage Guide
- Open your browser and go to `http://localhost:3000`.
- Use the provided interface to register attendance by filling out the required fields and submitting the form.
- You can view registered attendance entries in the provided section of the application.
